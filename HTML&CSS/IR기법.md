# IR 기법

### IR 기법이란?
'Image Replacement'의 줄임말로, 이미지를 볼 수 없는 사용자에게 적절한 대체 텍스트를 제공하는 것을 의미

##### 사용하는 경우:
- <img> 태그 사용 시, 대체 텍스트가 너무 긴 경우
- background-image 속성으로 처리한 의미 있는 이미지

##### 사용 방법: 
- 시멘틱 마크업 유지
- 스크린 리더 인식 가능하고, 레이아웃과 성능에 영향이 없는 CSS 속성 사용
- .hidden 클래스 활용
